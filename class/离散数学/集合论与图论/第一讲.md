# 引言
## 集合论的主要内容
* 研究对象
```
    集合、关系、函数、自然数、基数
```
* 研究思想
```
    以逻辑为基础，以集合为工具,表示和构造各种数学对象
```
* 研究内容
```
    1.集合的基本概念：集合之间的关系、运算、恒等式
    2.二元关系：表示、性质、函数、等价关系、序关系
    3.自然数：皮亚诺系统、自然数的运算、性质
    4.基数：有穷集与无穷集、基数的比较
    5.序数：良序、超限归纳法
```
## 图论的主要内容
* 研究对象
```
    有顶点和边构成的图
```
* 研究思想
```
    以集合论为基础、以图为工具、为各种二元关系建立模型
```
* 研究内容
```
    1.图的基本概念：连通性、矩阵的表示、带权图
    2.欧拉图、哈密顿图：边和顶点的遍历
    3.树：表示层级组织关系
    4.平面图：判定、表示、性质
    5.图的着色：各种调度问题的模型
    6.独立集、支配集、覆盖集、匹配：各种应用问题
```
